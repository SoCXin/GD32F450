  @brief
    This demo is based on the GD32150R-EVAL board, it shows how to use the USART
  to wake up the MCU from DeepSleep mode.
    In this demo, the wake up Method is configured as Start bit dection.When the MCU
  enter into DeepSleep mode,the LED2 stop in a certain stateis(on or off).In this case,
  the LED2 go on blink if USART1 was wake up.
