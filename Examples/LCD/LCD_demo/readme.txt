  @brief
    This demo is based on the GD32190R-EVAL board, it shows how to use the LCD
  peripheral to work with glass LCD.
    For the reason LCD and RTC share the same clock, we must configure the clock
  of RTC first, this demo use LSI(low-speed internal) to drive the LCD peripheral.
    In this demo, the LCD peripheral use 1/3 Bias voltage, 1/4 duty, 156.25Hz update
  frequence to display numbers.
