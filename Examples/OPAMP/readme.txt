  @brief
    This demo is based on the GD32190R-EVAL board, it shows how to use the operational 
  amplifier to amplify the input voltage, input voltage connected to Vin(JP8),We can
  use multimeter or oscilloscope on Vout(JP8).While, can use USART2 Printf to the serial
  port assistant.
