  @brief
    This demo is based on the GD32150R-EVAL/GD32190R-EVAL board,it shows how to 
  use one DACl to generate Analog power supply on DAC Channel1 output.We can 
  use multimeter or oscilloscope on PA4,(JP1) on the GD32150R-EVAL,The voltage 
  value is 1.656V.(JP5) on the GD32190R-EVAL.The voltage value is 2.502V.
