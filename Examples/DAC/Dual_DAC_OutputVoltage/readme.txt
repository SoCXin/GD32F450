  @brief
    This demo is based on the GD32190R-EVAL board, it shows how to use Dual DAC 
  to generate Analog power supply on Dual DAC output.We can use multimeter or 
  oscilloscope on DAC_OUT1 and DAC_OUT2(JP5).The voltage value of DAC_OUT1 (JP5) 
  is 0.624V,the voltage value of DAC_OUT2 (JP5) is 2.498V.



  
  Jumper cap (JP6) should be on 1 and 2.
