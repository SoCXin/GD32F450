  @brief
    This demo is based on the GD32150R-EVAL/ GD32190R-EVAL board, on the GD32150R-EVAL,
  Tamper key connected to PC13.LED1 connected to PC10. on the GD32190R-EVAL, 
  Tamper key connected to PC13.LED1 connected to PA11. It provides a description 
  of how to use the key to control the LED. Press down the Key, LED1 will be lighted. 
  Press down the Key again, LED1 will be off.
