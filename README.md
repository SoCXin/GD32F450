####  qitas@qitas.cn
#### 父级：[vendor](https://github.com/sochub/vendor) 

GigaDevice北京兆易创新，成立于2005年4月，是国内首家专业从事存储器及相关芯片设计的集成电路设计公司。

GD32系列产品兼容STM32相同规格产品，但主频提高约50%，可以相互替换，但在软件资源兼容性上还有些许不足。
